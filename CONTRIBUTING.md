# Contributing to Social Automation

Thank you for your interest in contributing to this project.

## Current Status

This is currently a personal project for automating faith-based TikTok content. While the repository is public for transparency and learning purposes, active external contributions are not being accepted at this time.

## If You'd Like to Contribute

If you have suggestions, bug reports, or feature ideas:

1. **Bug Reports**: Open an issue describing the problem, steps to reproduce, and expected behavior
2. **Feature Requests**: Open an issue describing the feature and its use case
3. **Questions**: Open a discussion or issue

## Code Standards

If contributions are accepted in the future, they would need to follow:

- Python 3.12+ with type hints
- Ruff for linting (all checks must pass)
- Pytest for testing (maintain >80% coverage)
- Follow the safety rules in CLAUDE.md

## Content Guidelines

Any content-related contributions must adhere to:

- No prosperity gospel or "name it and claim it" language
- No manipulative, fear-based, or guilt-based messaging
- Reverent, respectful, and encouraging tone
- Accurate Bible verse text per the specified translation
